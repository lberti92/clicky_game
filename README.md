# Clicky-Game (React.js)

### About

* `CLICKY-GAME` is a memory game built with `React.js` The goal of the game is simple, using your mouse, `CLICK` each of the twelve `Minions` character cards in whatever order you chose - the order does not matter, only that you do not click the same card twice!  

* `EACH` click triggers the board of cards to reshuffle, thus, you cannot simply click the cards in the order in which they appear on screen.

* `COUNTERS` displayed to keep track of your current as well as top score.

* `CLICK` your way to the selecting all of the `Minions`!

### Summary

* `CLICKY-GAME` is built with `React.js`. Thus, the application is component based, able to keeps track of change events using `state`, and updates to the page render dynamically and without page refresh.
